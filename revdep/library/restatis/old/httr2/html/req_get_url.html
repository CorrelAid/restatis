req_get_url {httr2}R Documentation

Get request URL

Description

Retrieve the URL from a request.

Usage

req_get_url(req)

Arguments

req

A httr2 request object.

Value

A character string.

Examples

request("https://httpbin.org") |>
  req_url_path("/get") |>
  req_url_query(hello = "world") |>
  req_get_url()
