# List of EVAS Codes

List of EVAS Codes

## Usage

``` r
evas_list
```

## Format

### `evas_list`

A data frame with 1132 rows and 3 columns:

- EVAS:

  EVAS code

- Beschreibung:

  Details on the EVAS code

- Titel:

  Alternative desription of EVAS code contents
